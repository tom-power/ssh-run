Run scripts against hosts over ssh

Usage:
  sshRun --hosts                
  sshRun <hostName> (--ssh|--scripts|--explain)
  sshRun <hostName> <scriptName> [--explain]
  sshRun --help

Options:
  --hosts         List hosts
  --scripts       List scripts for host
  --explain       Explain the host or script
  --help          Show this screen
